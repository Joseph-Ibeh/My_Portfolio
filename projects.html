Weather Dashboard

A responsive weather dashboard that provides current weather information based on user input. It includes features to display temperature, weather conditions, humidity, and wind speed. Designed to be user-friendly with a clean interface and responsive design.

  • Users can enter the name of a city to get current weather data.
  • Displays temperature, weather description, humidity, and wind speed.
  • Includes error handling for cases where the city is not found or if there are issues with the API request.
  • Responsive Design ensures the dashboard works seamlessly on mobile devices, tablets, and desktops.
  • Uses the OpenWeatherMap API to fetch real-time weather data.
